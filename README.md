# 🐍 Snake Game (HTML, CSS, JavaScript)

A fun and interactive Snake Game built using **pure HTML, CSS, and JavaScript**.
Includes modern UI, on-screen controls, and a restart system.

---

## 🎮 Features

* 🐍 Classic Snake gameplay
* 🎮 Keyboard + On-screen button controls
* 📱 Mobile-friendly design
* 💀 Game Over screen with restart option
* 🌈 Smooth UI with glowing effects
* ⚡ Lightweight and fast (no libraries used)

---

## 🚀 Demo

👉 Open the `snake.html` file in your browser to play the game.

---

## 🕹️ Controls

### Keyboard:

* ⬆️ Arrow Up → Move Up
* ⬇️ Arrow Down → Move Down
* ⬅️ Arrow Left → Move Left
* ➡️ Arrow Right → Move Right

### On-Screen Buttons:

* Tap buttons to control the snake (works on mobile)

## 🧠 How It Works

* The game uses an HTML `<canvas>` to render the snake and food.
* JavaScript controls:

  * Snake movement
  * Collision detection
  * Score tracking
* CSS adds styling and animations.
